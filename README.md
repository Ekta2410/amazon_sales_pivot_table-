# amazon_sales_pivot_table-
1. Cleaned the dataset by removing rupee symbols, percentage signs, extra spaces, and handling blank values.
2. Converted price and discount columns into numeric format.
3. Created Pivot Tables to analyze:
   - Average discounted price by category
   - Number of products by category
   - Average ratings by category
4. Applied sorting and conditional formatting to highlight top-performing categories.
5. Added slicers for category and rating to make the analysis interactive.
6. Created Pivot Charts to visually compare category performance.
7. Derived business insights based on pivot table analysis.
8. Exported the final report in Excel and PDF formats.
